
# Proyecto Entorno Cliente 01

Este proyecto es una onepage estática frontend que incluye una estructura sencilla de HTML, CSS y JavaScript. El proyecto se organiza con una página principal, estilos CSS y un script para animar elementos del DOM.

## Tecnologías utilizadas

Este proyecto ha sido desarrollado utilizando las siguientes tecnologías y herramientas:

- HTML5 / CSS3
Lenguajes de marcado y estilos utilizados en la estructura y presentación de las páginas.

- Font Awesome
Librería de íconos utilizada para elementos visuales en la interfaz

- JavaScript
Utilizado para animar elementos del DOM.

- Git & GitHub
Control de versiones y alojamiento del repositorio.

## Estructura

| Elemento | Descripción |
|-----------|-------------|
| `/` | Directorio raíz del proyecto |
| `index.html` | Página principal del proyecto (frontend estático) |
| `css/` | Carpeta con estilos CSS personalizados |
| `fontawesome/` | Carpeta con assets de íconos Font Awesome |
| `img/` | Carpeta con imágenes utilizadas en la web |
| `script.js` | Archivo JavaScript principal que contiene la lógica de interacción del sitio |
